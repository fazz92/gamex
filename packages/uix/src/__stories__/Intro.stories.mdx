# Basic Design System

Design System is a reusable component library that abstracts component abstraction based on baseui. Also fully exports baseui implementation.

## Install

Design System components are written in React, and its stories are written in [Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df).

Add Design System to your project.

`yarn add @gamex/uix`

## Use

Import components you want into your UI

```javascript
import { Input } from '@gamex/uix/lib/input'
```

and use them like so

```jsx
const example = () => (
  <div>
    <Input placeholder="Some Placeholder"/>
  </div>
)
```
